# 《剑来》经典桥段仿写

> 学习目标：掌握仙侠的意境描写、道理讲述、烽火文笔技巧

---

## 桥段一：陈平安第一次出剑（道理之剑）

### 原桥段概述
陈平安的剑不是普通的剑，是"道理剑"。他的每一剑都在讲道理。这个桥段展现了《剑来》最核心的设定——剑来，就是讲道理。

### 结构拆解
```
1. 场景设定：有人欺负弱者
2. 陈平安出现：安静地看着
3. 对方挑衅：嘲笑陈平安
4. 陈平安出剑：每一剑都在讲道理
5. 结局：道理讲完了，敌人认输
```

### 我的仿写版本
```
那天，我在街上看到一个人被欺负。

那个人跪在地上，满脸是血，而打他的人站在他面前，得意洋洋。

"我说了，欠债还钱，天经地义！"那人冷笑，"不还钱，就别想有好果子吃！"

我没有说话，只是走过去，站在他们中间。

"你谁啊？"那人看了我一眼，"少管闲事！"

"我不是来管闲事的。"我说，"我只是来跟你讲道理的。"

"讲道理？"他笑了，"拳头就是道理！"

我没有说话。

我只是抬起了手。

然后，我的剑，来了。

"第一剑。"我的剑指着他，"欠债还钱，是道理。但不是让你打人的道理。"

我一剑挥出，他的衣服被剑气割破。

"第二剑。"我再挥一剑，"你已经打了他，这是错的道理。"

他的脸上多了一道血痕。

"第三剑。"我的第三剑，架在他脖子上，"现在，你有两个选择。"

"第一，把钱还给他，然后滚。"

"第二，不还钱，然后我让你滚。"

他看着我，眼中满是恐惧。

"我...我还钱！"

我收剑，转身离开。

身后，那人已经吓得瘫倒在地。

这就是道理。

不需要拳头。

只需要一剑一剑地讲。

等他听完，他就懂了。

而我的剑，从不骗人。

因为——

剑来。
```

### 技巧笔记
- 道理剑：每一剑都在讲道理
- 简洁对话：不多废话
- 行动说明：先破坏再威胁
- 结尾呼应："剑来"

## 桥段二：陈平安守剑气长城（悲壮担当）

### 原桥段概述
陈平安在剑气长城守护，不为名利，只为身后的人。这是全书最悲壮的场景之一。

### 结构拆解
```
1. 背景：蛮荒世界入侵
2. 众人退缩：陈平安站出来
3. 守护：站在最前面
4. 孤独：没有人理解
5. 坚持：继续守护
```

### 我的仿写版本
```
剑气长城上，所有人都退了。

敌人百万大军压境，而守军只有一万。

"退了！我们退了！"有人大喊。

我站在城头，看着远方的敌军。

没有人要求我留下。

但我留下了。

"陈平安，你疯了吗？"有人问我。

"没有。"我说。

"你不怕死？"

"怕。"我老实说，"但我更怕——"

"怕我退了，身后的人就得死。"

我没有再解释。

我转身，站在城头最前方。

敌人冲来了。

我没有动。

我只是在等。

等我的剑来。

"剑来。"

那一刻，万剑齐发。

我身后，无数把剑飞向天空。

那是剑气长城上，所有剑修的剑。

它们响应了我的呼唤。

那一刻，我仿佛看到了齐先生。

他站在我身边，对我笑。

"平安啊，你做得很好。"

我握紧剑，冲入敌阵。

我不知道我能不能守住。

但我知道——

我身后，是我必须守护的人。

哪怕只有我一个人。

我也要站在这里。

因为我是——

剑气长城的守门人。

而守门人——

永不后退。
```

### 技巧笔记
- 悲壮感：一个人vs百万大军
- 道理：不是为名利，只为身后的人
- 万剑齐发：剑来
- 精神传承：齐先生的出现
