# 《神墓》经典桥段仿写

> 学习目标：掌握辰东的神秘世界观、复活设定、神墓流套路

## 桥段一：辰南复活（神魔陵园）

### 原桥段概述
辰南在神魔陵园醒来，不知道自己是谁，周围都是神魔的坟墓。这个开篇奠定了全书神秘、悲壮的基调。

### 结构拆解
```
1. 醒来：辰南在神魔陵园醒来
2. 迷茫：不知道自己是谁
3. 探索：发现神魔坟墓
4. 觉醒：发现自己有力量
5. 决心：开始寻找身份
```

### 我的仿写版本
```
我醒了。

但我不知道我是谁。

我躺在一个巨大的陵园中，周围全是坟墓。

每一座坟墓，都散发着强大的气息。

"这里是..."我喃喃道。

神魔陵园。

埋葬神和魔的地方。

而我，竟然在这里醒来了。

我坐起身，看着周围的坟墓。

有神的坟墓，有魔的坟墓。

他们曾经都是这个世界上最強的存在。

但现在，他们都死了。

只有我，活着。

"我是谁？"我问自己。

没有人回答我。

我只知道，我必须离开这里。

我必须找到我是谁。

我必须找到我的过去。

这就是我活下去的理由。

也是我修炼的意义。

神墓之中，我辰南——

重生了。
```

### 技巧笔记
- 神秘感：神魔陵园的设定
- 迷茫感：不知道自己是谁
- 决心：寻找身份
- 重生感：辰南的复活
