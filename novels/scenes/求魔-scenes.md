# 《求魔》经典桥段仿写

> 学习目标：掌握耳根的悲壮基调、求魔理念、宿命设定

---

## 桥段一：苏铭的求魔之路（核心理念）

### 原桥段概述
苏铭在修炼道路上选择了"求魔"之路，而不是传统的修仙。这个理念打破了正邪之分，是全书最核心的设定。

### 结构拆解
```
1. 迷茫：苏铭不知道该走什么路
2. 顿悟：苏铭领悟了求魔的理念
3. 选择：苏铭选择求魔
4. 坚持：苏铭坚持自己的道路
```

### 我的仿写版本
```
"什么是魔？"

我站在山崖上，看着远方的天空。

正道说，魔是邪恶的。

正道说，修仙才是正道。

但我不这么认为。

魔，不是邪恶。

魔，只是一种力量。

一种不被束缚的力量。

"我选择求魔。"我说。

师父看着我，眼中有些担忧。

"苏铭，求魔之路，艰难万险。你确定？"

"我确定。"我说。

"为什么？"

"因为——"

"我不想被束缚。"

"我不想被所谓的正道束缚。"

"我只想走自己的路。"

"这，就是我的道。"

"求魔之道。"

从那天起，我踏上了求魔之路。

正道说我是魔。

但我不在乎。

因为——

魔，不是贬义。

魔，是我自己的选择。

这就是苏铭。

求魔之人。

永远不被束缚。

永远走自己的路。
```

### 技巧笔记
- 求魔理念：打破正邪之分
- 坚持自己：不被束缚
- 独立宣言："我不想被束缚"
- 耳根风格：悲壮但坚定
