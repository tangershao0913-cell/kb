# 《莽荒纪》经典桥段仿写

> 学习目标：掌握莽荒世界的宏大设定、纪宁的成长线

---

## 桥段一：纪宁父亲失踪（悲情开场）

### 原桥段概述
纪宁的父亲是纪部的英雄，但在某次战斗中失踪了。纪宁发誓要找到父亲，这是全书最核心的主线之一。

### 结构拆解
```
1. 父亲是英雄：纪宁崇拜父亲
2. 失踪：父亲在某次战斗中消失
3. 悲痛：纪宁和母亲陷入悲痛
4. 决心：纪宁发誓找到父亲
5. 修炼：纪宁开始修炼
```

### 我的仿写版本
```
我的父亲，是纪部的王。

他高大、威武，是纪部最强的战士。

从小，我就跟在他身后，看着他带领族人南征北战。

"父亲，你是最强的！"我仰着头说。

父亲笑了，摸摸我的头："纪宁，以后你也会这么强的。"

那天，父亲接到了一个任务。

敌人入侵，他必须出征。

"父亲，你一定要回来！"我拉着他的手。

"放心，父亲一定会回来。"

他走了。

这一走，就是三年。

三年后，战争结束了。

但父亲，没有回来。

"纪宁，你父亲他..."回来的士兵欲言又止。

"我父亲怎么了？！"我急切地问。

"失踪了。在最后一场战斗中，他消失在了战场上。"

那一刻，我的世界崩塌了。

我跑回房间，关上门，一个人哭了很久。

从那天起，我再也没有见过父亲的笑容。

但我没有放弃。

"父亲，我一定会找到你。"

我站在纪部的大殿上，对着天空发誓。

"不管你在哪里，我都会找到你。"

"哪怕走遍整个莽荒世界。"

"哪怕走遍整个三界。"

"我，纪宁，一定会找到你！"

从那天起，我开始疯狂地修炼。

我要变强。

强到有一天，可以去找父亲。

这是我活下去的理由。

也是我修炼的意义。
```

### 技巧笔记
- 父亲形象：先建立父亲的伟大
- 悲剧设定：失踪而非死亡，留下希望
- 决心：纪宁发誓找到父亲
- 目标明确：推动全书主线

---

## 桥段二：纪宁修炼（成长）

### 原桥段概述
纪宁开始修炼，一步步变强，最终成为三界的顶级强者。

### 结构拆解
```
1. 起点：纪宁开始修炼
2. 困难：遇到各种挑战
3. 突破：纪宁不断突破
4. 成长：一路升级
5. 变强：最终成为强者
```

### 我的仿写版本
```
修炼的第一天，师父说我没有天赋。

"纪宁，你的修炼天赋太差了。"师父摇头，"可能不适合修炼。"

我没有说话。

我只是继续修炼。

别人修炼一个时辰，我修炼三个时辰。

别人休息的时候，我还在修炼。

一年后，我的修为超过了同门师兄。

三年后，我成为了宗门最年轻的筑基期。

十年后，我成为了宗门的长老。

而我的目标，从来只有一个——

找到父亲。

师父问我："纪宁，你为什么这么拼命？"

我说："因为我在找人。"

"找谁？"

"找我父亲。"

师父沉默了很久。

然后他说："纪宁，你会找到他的。"

我笑了。

"我一定会。"

这就是我修炼的理由。

不是为了变强。

不是为了名利。

只是因为——

我要找到父亲。

这是我活着的意义。

也是我修炼的意义。
```

### 技巧笔记
- 努力型主角：天赋不够，努力来凑
- 目标明确：一直是为了找父亲
- 成长展现：时间跨度，稳步提升
- 简洁对话：不用多解释

## 核心套路总结

| 桥段类型 | 情绪功能 | 核心公式 |
|---------|---------|---------|
| 父亲失踪 | 悲情/决心 | 父亲形象→悲剧→发誓 |
| 修炼成长 | 坚定/期待 | 困难→努力→突破→成长 |
