# 《遮天》经典桥段仿写

> 学习目标：掌握九龙拉棺开篇、荒古圣体设定、辰东式战斗描写

---

## 桥段一：九龙拉棺（最强开篇）

### 原桥段概述
叶凡与同学在泰山旅游时，九龙拉棺突然从天空降临，将众人带入星空。这是网文历史上最具创意的开篇之一，九条真龙拉着棺材从天而降，震撼感无与伦比。

### 结构拆解
```
1. 场景设定：泰山之巅，同学聚会
2. 日常铺垫：叶凡与老同学聊天
3. 异变发生：天空突然变暗
4. 九龙拉棺：九条真龙拉着棺材从天而降
5. 被拉入棺：众人被卷入棺材
6. 进入星空：众人开始星际旅行
7. 到达北斗：故事正式开始
```

### 我的仿写版本
```
那天，我们几个老同学约在泰山聚会。

毕业十年，大家各自奔忙，难得聚一次。

"叶凡，你现在在做什么？"老班长问我。

"没什么，就是个普通上班族。"我笑了笑。

"唉，当年在学校你最聪明，没想到......"他欲言又止。

我知道他想说什么。

当年我是全班第一，现在却泯然众人。

我没有说话，只是看向远方的天空。

忽然，天色暗了下来。

"怎么回事？要下雨了？"老班长抬头看天。

我也抬头。

然后，我看到了这辈子最震撼的一幕。

九条巨龙！

每一条都有山那么大，在云层中穿梭！

而在它们身后，是一口巨大的棺材！

"这...这不可能......"老班长已经说不出话来。

周围所有人都愣住了，有人甚至跪了下去。

"神仙显灵了！"

"那是龙！真龙！"

在众人的惊呼声中，九龙拉棺，向我们俯冲而来！

我还没来得及反应，就被一股无形的力量卷入了棺材之中。

等我再睁开眼，我已经不在泰山了。

我躺在一片陌生的土地上。

天空是紫色的。

有两个太阳。

"这是......哪里？"我喃喃自语。

没有人回答我。

但我知道，从这一刻起，我的人生彻底改变了。

因为九龙拉棺，我离开了地球。

来到了一个全新的世界。

北斗星域。

一个属于修士的世界。

而我，叶凡，将在这里开始全新的冒险。
```

### 技巧笔记
- 场景对比：日常同学聚会 vs 九龙拉棺从天而降
- 震撼感：九条龙+棺材的视觉冲击
- 日常感：叶凡的"失败"与同学们的"惋惜"——后面逆袭打脸更爽
- 悬念：两个太阳的紫色天空——新世界正式开始
- 简洁有力：最后一句直接进入主题

## 桥段二：荒古圣体（最强体质觉醒）

### 原桥段概述
叶凡在北斗星域发现自己竟然是荒古圣体——一种传说中无敌的体质，但因为天地变化，现在被认为是"废体"。这个设定是全书最核心的设定，也是叶凡逆袭的基础。

### 结构拆解
```
1. 被嘲笑：叶凡被认为是废体
2. 测试：各大势力测试叶凡
3. 震惊：发现是荒古圣体
4. 失望：荒古圣体已经没落
5. 希望：叶凡决定修炼
6. 转折：叶凡的圣体与众不同
```

### 我的仿写版本
```
"荒古圣体？"

当我听到这个词的时候，心跳加速。

"没错，你是荒古圣体。"那位老者看着我，眼中满是震惊，"这是传说中最强大的体质之一。"

"最强体质？"我眼睛一亮，"那我岂不是天才？"

老者摇了摇头。

"曾经是。"

"曾经？"

"荒古圣体在荒古时代是无敌的体质，但后来天地大变，这种体质无法修炼，被认定为废体。"

我的心沉了下去。

"也就是说......我是废体？"

老者沉默片刻，然后点头。

"在现在这个时代，荒古圣体确实无法修炼。你身上的资源再多，也很难突破。"

周围传来嘲笑声。

"哈哈哈，荒古圣体？在这个年代就是废物！"

"还以为是什么天才，原来是个笑话。"

我握紧拳头，没有说话。

但我没有放弃。

既然这是天地变化造成的，那我就打破这个限制。

既然这个时代无法修炼荒古圣体，那我就创造一个新的修炼方法。

老者看着我，眼中闪过一丝光芒。

"你有信心？"

"没有。"我老实回答，"但我不想认输。"

老者笑了。

"好，有骨气。既然如此，老夫就送你一场机缘。"

他从怀中取出一枚金色的果实。

"这是菩提子，是修炼荒古圣体的关键之一。"

"有了它，你的圣体或许能再次觉醒。"

我接过菩提子，心中燃起希望。

"谢谢前辈。"

老者摆摆手，转身离开。

"希望下次见面时，你已经不再是废物。"

我看着他离去的背影，暗暗发誓。

总有一天，我会让所有人知道——

荒古圣体，从未没落！

只是你们不懂而已！
```

### 技巧笔记
- 悬念→反转→失望→希望：情绪层层递进
- 老者的态度：从震惊→失望→欣赏——增加层次感
- 主角心态：不认输，即使被认定为废体
- 转折点：菩提子——金手指出现
- 霸气宣言："荒古圣体，从未没落"——坚定有力

## 桥段三：叶凡vs摇光圣地（经典战斗）

### 原桥段概述
叶凡在四极秘境时，挑战摇光圣地的天才弟子。这是全书最经典的战斗之一，展现了叶凡的战斗智慧和荒古圣体的威力。

### 结构拆解
```
1. 挑衅：摇光圣地弟子嘲笑叶凡
2. 应战：叶凡接受挑战
3. 战斗开始：双方交手
4. 危机：叶凡落入下风
5. 爆发：荒古圣体之力爆发
6. 碾压：叶凡反败为胜
7. 打脸：摇光弟子震惊
```

### 我的仿写版本
```
摇光圣地的弟子看着我，脸上带着嘲讽。

"荒古圣体？"他笑了，"在这个时代，荒古圣体就是废物。"

周围的人也跟着笑起来。

我没有说话，只是看着他。

"怎么？不服气？"他更加得意，"我来给你一个机会。跟我打一场，赢了，我给你十万斤源。输了......"

"输什么？"

"输了，你当众跪下，叫我一声爷爷。"

他的话音刚落，我就动了。

"好。"

我只有这一个字。

然后，我的拳头已经出现在他面前。

"你——！"他大惊，连忙抵挡。

但已经来不及了。

我的拳头砸在他身上，他整个人倒飞出去！

"怎么可能？！"他在空中惊呼。

我脚下不停，追上去，又是一拳！

"砰！"

他重重摔在地上，口吐鲜血。

"你...你的力量......"他满脸不可置信。

"荒古圣体，从不没落。"我站在他面前，俯视着他，"你不懂，只是你太弱了。"

周围一片死寂。

所有人都震惊地看着我。

摇光圣地的弟子，竟然被一个"废体"打败了。

我蹲下身，看着他。

"十万斤源，现在给我。"

他的脸色煞白。

"你...你敢抢摇光圣地的东西？"

"这是你赌的。"我笑了，"我只是拿回属于我的东西。"

我站起身，对周围的人说：

"今天的事，大家都看到了。"

"从今以后，谁再说荒古圣体是废体——"

"这就是下场。"
```

### 技巧笔记
- 快速战斗：不拖泥带水，直接一拳
- 打脸节奏：前面多嚣张，后面多惨
- 霸气宣言："荒古圣体，从不没落"——直接成为全书金句
- 简洁收尾：不需要多解释，结果说明一切
