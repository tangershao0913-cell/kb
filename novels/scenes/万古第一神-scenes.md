# 《万古第一神》经典桥段仿写

> 学习目标：掌握风青阳的爽文套路

---

## 桥段一：李天命觉醒（废柴逆袭）

### 原桥段概述
李天命从废物逆袭成为万古第一神。

### 结构拆解
```
1. 被嘲笑：被认定为废物
2. 获得传承：获得神秘金手指
3. 觉醒：开始修炼
4. 打脸：让嘲笑者震惊
```

### 我的仿写版本
```
"李天命，你就是个废物！"

这句话，我听了十年。

我是李家主的儿子，但我无法修炼。

在这个世界，无法修炼就是废物。

但今天，一切都将改变。

那天，我在后山获得了一本神秘古籍。

"这是一部逆天的功法！"我兴奋地说。

我开始修炼。

一天后，我就突破了！

一周后，我就超过了同门师兄！

一个月后，我就成为了家族第一！

所有人都震惊了。

"这...这怎么可能？！"他们说。

我笑了。

"我说过，我是废物？"

"你们，真的确定？"

从今天起，我李天命——

万古第一神！
```
