# 《武动乾坤》经典桥段仿写

> 学习目标：掌握天蚕土豆的爽文套路

---

## 桥段一：林动获得石符（金手指）

### 原桥段概述
林动在矿山中获得神秘石符，这是他逆袭的关键。

### 结构拆解
```
1. 场景：林动在矿山
2. 发现：发现神秘石符
3. 认主：石符与林动融合
4. 觉醒：林动开始修炼
```

### 我的仿写版本
```
我在矿山中找到了一个石符。

它看起来很普通，但我感觉到了它的不凡。

我触碰了它。

刹那间，一股力量涌入体内！

"这是...石符？！"我惊喜道。

石符可以吸收天地能量，召唤符兽！

从此，我就是林动！

武动乾坤！
```
