# 《斗破苍穹》大纲

---

## 整体结构

**总字数**：约533万字
**章节数**：约1600章
**核心主线**：萧炎从废柴到斗帝的逆袭之路

---

## 起（1-300章）：废柴觉醒，复仇之约

### 第一卷：乌坦城篇（1-100章）

**章节核心事件**：
- 萧炎被纳兰嫣然退婚，立下三年之约
- 药老现身，焚诀传承
- 萧炎开始修炼，进步神速
- 炼药师天赋觉醒
- 进入迦南学院

**情节线**：
```
第1-10章：退婚之辱，立三年之约
         → 核心：莫欺少年穷，情绪共鸣点
第11-30章：药老觉醒，焚诀传承
         → 核心：金手指出现，目标明确
第31-60章：修炼崛起，小试牛刀
         → 核心：打脸嘲讽者，爽点密集
第61-80章：迦南学院入学
         → 核心：新地图，新人物，新冲突
第81-100章：学院大比
         → 核心：第一次真正实力展示，碾压
```

**起承转合**：
- **起**：废柴被羞辱
- **承**：金手指出现，开始修炼
- **转**：学院大比展现实力
- **合**：三年之约时间到，准备赴约

---

### 第二卷：三年之约篇（100-200章）

**章节核心事件**：
- 萧炎实力飞速提升
- 炼药术超越同龄人
- 云岚宗追杀萧家
- 三年之约到来
- **萧炎击败纳兰嫣然** ← 全书第一个大型高潮
- 云岚宗大战
- 击杀云山，灭云岚宗

**高潮设计**：
```
三年之约战斗结构：
├── 第1回合：纳兰嫣然使出云岚宗剑法（占上风）
├── 危机时刻：萧炎斗气耗尽（读者紧张）
├── 底牌出现：药老加持+青莲地心火（逆转）
├── 结果：萧炎完胜，纳兰嫣然心服口服
└章末：萧炎名震加玛帝国
```

**起承转合**：
- **起**：赴约之路，云岚宗阻挠
- **承**：三年修炼成果展示
- **转**：纳兰嫣然使出全力
- **合**：击败纳兰嫣然，灭云岚宗

---

## 承（300-800章）：大陆闯荡，势力积累

### 第三卷：黑角域篇（200-350章）

**核心事件**：
- 离开加玛帝国，进入黑角域
- 收服海波东
- 炼制丹药，积累财富
- 创建炎盟
- 第一次与魂殿交锋

**地图设计**：
- 黑角域 = 混乱之地，无法无天
- 特点：没有规则，实力为尊
- 萧炎在这里积累资源，升级实力

---

### 第四卷：中州篇（350-600章）

**核心事件**：
- 进入中州
- 寻找父亲
- 组建势力
- 收集异火
- 与魂殿正面对抗

**中州特点**：
- 顶级势力云集
- 斗宗多如狗，斗尊满地走
- 萧炎在这里不断越级挑战

---

### 第五卷：异火篇（600-800章）

**核心事件**：
- 收集多种异火
- 焚诀不断进化
- 势力越来越大
- 药老被魂殿抓走 ← 重大危机
- 救药老

**异火收集时间线**：
```
第1种：青莲地心火（乌坦城）
第2种：骨灵冷火（药老传承）
第3种：陨落心炎（迦南学院）
第4种：海心焰（黑角域）
第5种：三千焱炎火（中州）
...（共23种）
```

---

## 转（800-1200章）：魂殿之战，巅峰之路

### 第六卷：魂殿大战篇（800-1000章）

**核心事件**：
- 魂殿全面追杀
- 多次生死危机
- 净莲妖火争夺战
- 陀舍古帝传承出现
- **萧炎成为斗帝** ← 全书最高潮

**升级曲线**：
```
斗宗 → 斗尊 → 半圣 → 斗圣 → 斗帝
  ↑        ↑              ↑
  救人    中州称霸    大结局
```

---

### 第七卷：最终决战篇（1000-1200章）

**核心事件**：
- 魂天帝觉醒
- 远古种族大战
- 陀舍古帝传承
- **萧炎vs魂天帝** ← 最终决战
- 大陆太平，萧炎成为斗帝

**结局设计**：
- 萧炎成为斗帝
- 与萧薰儿结婚
- 药老团聚
- 大陆和平
- **HE（Happy Ending）**

---

## 合（1200-1600章）：收尾与番外

### 章节核心事件：
- 完善结局，收尾各条线
- 生子后的番外
- 各角色结局交代
- 彩蛋：萧炎儿子出生（对应《斗罗大陆》）

## 大纲核心公式

```
开局：废柴+退婚+三年之约
↓
金手指：老爷爷+焚诀+异火
↓
目标：三年之约 → 救师父 → 救父亲 → 成为斗帝
↓
爽点节奏：
├── 每50章：小副本胜利
├── 每200章：阶段性大高潮
└章均：3-5章一个小高潮
↓
地图推进：乌坦城 → 黑角域 → 中州 → 斗帝大陆
↓
反派升级：纳兰嫣然 → 云岚宗 → 魂殿 → 魂天帝
↓
结局：HE，萧炎成神
```
