# 《庆余年》大纲

---

## 整体结构

**总字数**：约370万字
**章节数**：约800章
**核心主线**：范闲从澹州少年到天下棋手的成长与复仇之路

---

## 起（1-150章）：澹州成长，京都搅局

### 第一卷：澹州篇（1-50章）

**章节核心事件**：
- 婴儿范闲醒来，拥有成年灵魂
- 五竹守护范闲成长
- 费介出现，教范闲毒术
- 范闲开始修炼无名功法
- 范老太太的养育之恩
- 赴京前的准备

**情节线**：
```
第1-10章：婴儿范闲，意识清醒
         → 核心：穿越设定开场，悬念拉满
第11-20章：五竹守护，神秘日常
         → 核心：机器人五竹的身份悬念
第21-35章：费介收徒，毒术传承
         → 核心：第一个师父，建立信任
第36-50章：澹州少年，武艺初成
         → 核心：告别澹州，准备赴京
```

**起承转合**：
- **起**：婴儿范闲，穿越重生
- **承**：五竹守护，费介授艺
- **转**：范闲发现自己的身世线索
- **合**：离开澹州，前往京都

---

### 第二卷：京都篇（50-150章）

**章节核心事件**：
- 庆庙偶遇林婉儿（一见钟情）
- 进入监察院，崭露头角
- 接手内库，接管鉴查院
- 牛栏街刺杀事件（滕子京之死）
- 范闲第一次杀人
- 出使北齐的准备工作

**高潮设计**：
```
牛栏街刺杀事件结构：
├── 背景：林婉儿与范闲订婚，长公主不满
├── 布局：林珙（林婉儿之兄）雇凶杀人
├── 危机：范闲遇刺，滕子京替死
├── 愤怒：范闲杀人，为滕子京报仇
├── 收尾：林珙被范闲设计杀死
└章末：范闲在京都站稳脚跟
```

**起承转合**：
- **起**：庆庙偶遇林婉儿
- **承**：接手内库，权力斗争
- **转**：牛栏街刺杀，第一次杀人
- **合**：滕子京之死推动范闲走上复仇之路

---

## 承（150-400章）：权力漩涡，身世之谜

### 第三卷：鉴查院篇（150-250章）

**核心事件**：
- 范闲成为鉴查院提司
- 与太子、二皇子的权力博弈
- 接手内库，面对长公主的刁难
- 春闱舞弊案
- 范闲与林婉儿的爱情发展
- 逐渐发现母亲叶轻眉的痕迹

**情节线**：
```
第150-180章：鉴查院接手
         → 核心：用现代管理学改革鉴查院
第181-210章：与太子党的冲突
         → 核心：范闲的棋局开始显露
第211-250章：叶轻眉的痕迹
         → 核心：监察院门口的石碑——"我希望庆国人民都能成为不羁之民"
```

---

### 第四卷：出使北齐篇（250-350章）

**核心事件**：
- 范闲出使北齐
- 北齐权臣沈重的博弈
- 肖恩的处置
- 海棠朵朵相识
- 战豆豆（北齐小皇帝，女扮男装）的秘密
- 肖恩临死前的秘密透露
- 范闲得知叶轻眉的部分真相

**高潮设计**：
```
北齐谈判结构：
├── 表面：外交对等，和平谈判
├── 暗线：范闲暗中操控北齐内斗
├── 危机：沈重设局，范闲险境
├── 破局：范闲策反沈重部下
├── 结果：北齐内乱，沈重被杀
└章末：范闲声望达到顶峰
```

---

### 第五卷：身世之谜篇（350-400章）

**核心事件**：
- 范闲发现自己可能是庆帝的儿子
- 陈萍萍开始向范闲透露叶轻眉的往事
- 范闲开始调查叶轻眉之死
- 二皇子的真面目暴露
- 叶流云出现
- 范闲与庆帝的第一次正面交锋

**核心悬念揭示**：
```
叶轻眉之死的线索：
├── 叶轻眉想建立人人平等的世界
├── 她拥有超越时代的财富（内库）
├── 庆帝忌惮她的影响力
├── 陈萍萍被庆帝囚禁
└章末：范闲确定——庆帝是杀害母亲的主谋
```

---

## 转（400-650章）：复仇之路，天下棋手

### 第六卷：复仇准备篇（400-500章）

**核心事件**：
- 范闲全面掌控鉴查院和内库
- 暗中培养势力
- 大东山祭祀（四大宗师齐聚）
- **大东山之战** ← 全书最精彩战斗
- 四顾剑和苦荷战死
- 庆帝身份暴露（最强的大宗师）

**高潮设计**：
```
大东山之战结构：
├── 表面：庆帝带范闲祭天
├── 阴谋：四顾剑+苦荷+叶流云联手杀庆帝
├── 真相：叶流云假装配合，实际是庆帝的人
├── 战斗：四顾剑和苦荷被庆帝重伤
├── 结果：两大宗师死亡，庆帝重伤
└章末：庆帝是大宗师，震撼全场
```

---

### 第七卷：陈萍萍之死篇（500-600章）

**核心事件**：
- 庆帝重伤，范闲成为天下最有权势的人
- 陈萍萍回到京都
- 陈萍萍调查叶轻眉之死的真相
- **陈萍萍被庆帝凌迟** ← 全书最虐场景
- 范闲发誓为陈萍萍报仇
- 范建替范闲挡刀，死亡

**催泪设计**：
```
陈萍萍之死结构：
├── 陈萍萍查出叶轻眉是被庆帝所杀
├── 陈萍萍进宫与庆帝对峙
├── 庆帝下令将陈萍萍凌迟
├── 范闲赶回京都，只来得及见最后一面
├── 陈萍萍临死遗言："她值得。"
└章末：范闲彻底与庆帝决裂
```

---

### 第八卷：最终决战篇（600-650章）

**核心事件**：
- 范闲与庆帝的决裂
- 太子、二皇子的结局
- 范闲逃离庆国
- 返回庆国，带回海棠朵朵等盟友
- **范闲vs庆帝最终决战**
- 范闲用巴雷特狙击枪重伤庆帝
- 庆帝被范闲击杀

**最终战斗模板**：
```
范闲vs庆帝结构：
├── 实力对比：范闲九品上 vs 庆帝大宗师
├── 第一阶段：范闲完全不是对手
├── 转折：范闲用巴雷特狙击枪重伤庆帝
├── 追击：范闲联合海棠朵朵、五竹围攻
├── 结局：庆帝被击杀
└章末：天下太平，范闲隐居
```

## 大纲核心公式

```
开局：穿越婴儿+五竹守护+身世悬念
↓
金手指：机器人五竹+毒术+巴雷特+前世记忆
↓
目标：查明母亲死亡真相 → 为母报仇 → 为陈萍萍报仇
↓
爽点节奏：
├── 权谋博弈：算无遗策，步步为营
├── 感情线：林婉儿是白月光
├── 复仇线：层层剥开庆帝的真面目
└章均：3-5章一个小高潮
↓
地图推进：澹州 → 京都 → 北齐 → 天下
↓
反派升级：林珙 → 二皇子 → 长公主 → 庆帝
↓
结局：HE，杀庆帝，为母报仇，隐居
```
